# Security Projects

Small cybersecurity tools and experiments for authorized lab environments.

## Structure

```text
projects/    Individual tools and experiments
templates/   Reusable project documentation
```

## Project Standards

- Keep each project self-contained with a README, usage notes, and dependencies.
- Prefer simple scripts and clear inputs over broad frameworks.
- Do not commit scan output, secrets, tokens, VPN files, private keys, or private target data.

## Usage Policy

Use everything in this repository only in legal, authorized environments.
